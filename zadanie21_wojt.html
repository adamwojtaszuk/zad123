Zadanie 21 – Użycie ID

Pojęcie Znaczenie (uzupełnione przez ucznia)
Co znaczy skrót ID? ID to identyfikator przypisany do pojedynczego elementu HTML w celu jego indywidualnego stylowania.
Jaki jest cel stosowania ID? Indywidualne nadanie stylu konkretnemu elementowi na stronie.
Jakie jest znaczenie znaku #? Znak # służy do odwołania się do ID w arkuszu stylów CSS.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Wywołanie ID następuje przez przypisanie atrybutu id="nazwa_id" w znaczniku HTML. W CSS odwołujemy się do niego poprzez #nazwa_id.
Uwagi na temat stosowania ID (trzy konieczne wyliczenia)
  • ID może być użyte tylko raz w dokumencie.
  • ID jest unikalne – nie powinno być powielane.
  • Do wielokrotnego stosowania lepiej użyć klasy (class).
